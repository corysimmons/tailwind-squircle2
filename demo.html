Tailwind Squircle Plugin

Create beautiful squircle shapes using SVG clip paths and the superellipse algorithm. Perfect for modern UI designs that need something between squares and circles.

Installation

npm install tailwind-squircle2

Then add the plugin to your tailwind.config.js:

module.exports = { content: ['./src/**/*.{html,js}'], plugins: [ require('tailwind-squircle2') ] }

Build your CSS:

npx tailwindcss build -o styles.css

Technical Details

Superellipse Formula

The plugin uses the superellipse (Lamé curve) formula in parametric form:

x = sign(cos(t)) × |cos(t)|^(2/n) y = sign(sin(t)) × |sin(t)|^(2/n)

Where n controls the shape:

  • n = 2: Ellipse/Circle
  • n = 3: Very rounded squircle
  • n = 4: Perfect squircle (balanced)
  • n = 6: Soft squircle (softer corners)
  • n = 8: Sharp squircle (more square-like)
  • n → ∞: Square
